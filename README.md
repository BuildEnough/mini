# Mini Project_TODO 리스트+ Chatbot 레이아웃

## 프로젝트 소개

`HTML, CSS, JavaScript`를 활용하여 진행한 미니 프로젝트입니다.  
브라우저의 `localStorage`를 이용해 데이터가 새로고침 이후에도 유지되도록 구현했으며,  
**기본적인 UI 구성**부터 **CRUD 기능**, **모달 형태의 채팅창**까지 단계적으로 완성했습니다.

## 주요 기능

### 1. ToDo List

- 할 일을 추가, 조회, 수정, 삭제할 수 있는 **CRUD 메모장** 구현
- `localStorage`를 활용하여 **새로고침 후에도 데이터 유지**
- 완료한 항목은 **체크 표시**로 상태 구분
- CSS를 이용해 **디자인 개선**
- HTML 파일 하나로도 동작 가능하도록 구성하여 **휴대폰 환경에서도 실행 가능**

### 2. 챗봇 레이아웃 만들기

- `document.createElement()`를 활용하여 **동적으로 채팅 UI 생성**
- 사용자와 챗봇의 대화창 레이아웃 구현
- `localStorage`를 사용하여 **새로고침 후에도 채팅 내역 유지**
- **대화내용 삭제 버튼** 추가

### 3. 최종 통합

- 우측 하단의 버튼을 누르면 **채팅창이 모달 형태로 열리도록 구현**
- 기존 ToDo List 및 챗봇 기능을 하나의 프로젝트로 통합
- 사용자가 보다 편리하게 사용할 수 있도록 인터페이스 개선

## 사용 기술

- **HTML**
- **CSS**
- **JavaScript**
- **localStorage**

이 프로젝트를 통해 다음 내용을 학습할 수 있었습니다.

- HTML 구조 설계

- JavaScript를 이용한 이벤트 처리
- `document.createElement()`를 통한 동적 요소 생성
